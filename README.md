News Management System
A dynamic web application designed to manage news content efficiently. This system allows administrators to handle news categories and articles through a secure dashboard.

🚀 Features
User Authentication: Secure login and logout system for administrators.

News Management: Complete CRUD operations to add, edit, and delete news articles.

Category Management: Ability to add and organize news into different categories.

Dashboard: A centralized control panel for managing all site content.

Media Uploads: Integrated system for uploading images and media related to news stories.

🛠️ Technical Stack
Language: PHP.

Database: MySQL (connected via db.php).

Web Tech: HTML, CSS, and Backend logic.

📂 Project Structure
login.php & register.php: Handle user access.

dashboard.php: The main admin interface.

add_news.php / edit_news.php / delete_news.php: Core news functionality.

uploads/: Folder to store news images.

📝 Localization
This project is built with a focus on Localization, making it easy to adapt for both Arabic and English-speaking users.
